# Core
The core of [Ido](https://github.com/ido-nvim), the ultra-extensible narrowing framework for Neovim.

## Basic usage
```vim
:lua require("ido").start{items = {"red", "green", "blue"}}
```

In place of `{"red", "green", "blue"}`, place the items you wish to narrow.
